    Fireworks vs Dreamweaver (Features and purpose)

    1. Fireworks is a graphics application intended for producing Web graphics.
      Whereas, Dreamweaver is a Web development tool that can combine both HTML and code to build websites using those programming codes.
    2. It is better to use Fireworks as a graphics and site-elements designing tool for the skeleton of the website.
      Whereas, Dreamweaver contributes to forming the overall build of the website.
    3. Both are used for different stages of a workflow process. In my project, I used Fireworks to first form the wireframe and storyboard graphics aspects.
      I then "sliced" the parts of the graphics and exported them to use in the Dreamweaver application.
    4. Although both are standalone applications, they both are empirical to the successful development of my homepage. Fireworks is intended for the focus of design elements,
      whereas Dreamweaver is used for the coding and structure of the overall website.
